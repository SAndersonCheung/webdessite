HTML: Hypertext Markup Language

What is HTML?

HTML is the programming language that makes up the Internet; every web page is made in a set type of HTML. Initially released in 1993, HTML has undergone many different changes. The latest release of HTML is HTML5

How does it work?

HTML functions through elements. Each HTML element is surrounded by a set of '< >' tags. The tags tell the page that it is an element, and the element tells the page how to display whatever it surrounds. For example, a functional type of HTML code would be:

<strong> This text is bold! </strong>

The element tags surround the text that the user wishes to make bold, and the word 'strong' defines that the user would like the text to be made bold.

Everything on a webpage will be made up of elements: some will be more complicated than others, but all are based on the same formula. Understanding the base formula is the first step to learning how to write HTML code!

How do I start a HTML document?

Note: typing '<!-- this -->' around your text, where 'this' is your text, makes the text a comment!

Every web page on the Internet must start with a <!DOCTYPE html> and must be closed by a </html>. Without this, the page will not be read as a HTML web page, and will be unreadable code to a compiler. The part of the page that is intended to be seen must be surrounded by <body> tags.

Code Line Purpose
<!DOCTYPE html> This tells the compiler that the file is a HTML file.
<html lang="en"> This tells the compiler that the file is a HTML file that is in English.
<meta charset="utf-8"> This marks the page as using charset 'utf-8', which replaced ASCII as the definition of characters on the Internet.
<title>Page Title</title> This defines the title of the page: what appears in the tab bar when the page is open.
<body> This marks the start of the 'body' - the page's actual content.
<h1>This is a Heading</h1> This defines a heading.
<p>This is a paragraph.</p> This defines a paragraph.
<p>This is another paragraph.</p> This defines a paragraph.
</body> This marks the end of the body segment.
</html> This marks the end of the HTML file.

What next?

Now that you have the base of your HTML document, you can easily fill it with text content using <h1> and <p> to your heart's content. However, the page looks a bit dull. Let's make it more interesting with some images and hyperlinks.

Note: You can have headers decreasing in size from <h1> all the way to <h6>!

<img src="one" alt="two">

With images, you don't need a closing tag, as they are self-contained. Now, let's decode that codebit. The one seen folowing 'src' is where the link to the image will be. This can be a link to an image online (using the URL after opening the image in a new tab), or an image contained within the file structure of the website. The two following 'alt' defines the text that will appear should the image fail to load whether due to an error, or the user having certain permissions denied.

<a href="three" title="four">This text would be the link!</a>

Links do use closing tags, and they make anything in between their tags the link, including images. The three following 'href' is the destination: where the link will take the user. The four following 'title' is an additional tooltip providing more information. It will be seen when the mouse hovers over the link.
