Resources

  • W3Schools

    For HTML, CSS, and JavaScript.

    W3Schools is a website that hosts a compilation of tutorials for all three web-development languages. The website is clean, simple, and easy to read and understand, as well as offering 'Try It Yourself!' pages allowing the user to attempt writing their own code using their tutorials. As such, it is a very useful resource for learning these languages, and is highly recommended by professionals in every capacity.

  • CodeAcademy

    For HTML, CSS, and JavaScript.

    CodeAcademy is a less specialist website that allows users to go through the motions of step-by-step learning a language. This is very good if you are teaching yourself the language from scratch, but not-so-useful if you come from an intermediate starting point, as you will be required to re-learn the basics.

  • Learn-HTML.org

    For HTML, and CSS.

    Learn-HTML.ORG is a website created specifically to aid in learning HTML and CSS. The main perk of this website over the previously listed websites for this is that it allows the user to try out code snippets without being taken to a seperate page, as it is on CodeAcademy.
