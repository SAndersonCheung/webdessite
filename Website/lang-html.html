Languages

In Website Design and Construction, there are three web development languages you will learn:

  • HTML
  • CSS
  • JavaScript

Each of these languages is essential for an aspect of website construction. The HTML (Hypertext Markup Language) is the bread of the website, where all of the text and base information is kept. The HTML of a website won't look pretty, but will be the functional aspect of the page. The CSS (Cascading Style Sheets) are connected to the HTML in the 'head' section. The CSS will change how the browser reads the HTML, making it appear much different than when it was pure HTML. Finally, JavaScript can be used to make the webpage much more interactive in a vast assortment of ways. From flashy, rotating image carousels, to support for everyday applications, such as flash applications and PDF's.
